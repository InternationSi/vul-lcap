# `使用指南`
需要启动所有子应用，找到主应用的入口

## 安装项目依赖
首先npm全局安装lerna,见官网 

安装依赖包方法
方法一. （建议使用第一种）
```
npm run bootstrap 建议使用淘宝镜像
```
方法二.
 cd进入项目文件夹 install安装依赖
### 启动admin应用
方法一. 在最外层
```
npm run admin 在项目文件直接运行,首屏加载较慢,稍作等待
```
方法二.需要cd进入相应文件后npm start （建议使用方法二）
